# Next.js / React 보안 취약점 패치 적용 보고서

**작성일자:** 2024년 12월 9일  
---

## 1. 취약점 개요

### CVE-2025-55182 (React Server Components)
- **위험도:** CVSS 10점 (최고 위험)
- **유형:** 원격 코드 실행 (RCE)
- **영향:** React Server Components에서 인증 없이 임의 코드 실행 가능
- **근본 원인:** 서버 측 프로토타입 오염 (Server-side Prototype Pollution)

### CVE-2025-66478 (Next.js)
- **위험도:** CVSS 10점 (최고 위험)
- **영향:** Next.js App Router 사용 시 취약
- **관련 패키지:** react-server-dom-webpack, react-server-dom-parcel, react-server-dom-turbopack

---

- 14.3.0-canary.77 이후 Canary 릴리스

### React
