# 공통사항

- 고객의 요구사항을 speckit 을 사용하여 요구사항을 분석하고 설계한다.
- 특별히 지시되지 않으면 고객의 요구사항은 1_prd/1_고객요구사항.md 의 문서의 내용이다.
- 별도의 지정된 파일이 있는 경우, 지정된 파일을 우선하여 사용한다
- 추가 지시된 파일이 있는 경우, 이을 분석하여 사용자 확인 후 진행한다

## 추가 파일

- 고객요구사항에 별도의 추가지시사항, 첨부문서가 있는 경우 이를 사용자에게 알리고 포함할지를 묻는다
- **설계 및 지시 충돌 시**: 설계 문서 간, 또는 지시 사항 간에 충돌이 발생하거나 기술적 문제가 발생하면 **반드시 사용자에게 확인 후 진행한다**. 임의로 판단하여 진행하지 않는다.
