# 프로젝트 implement 진행지침 (ctx.implement.md)

- 한글로 진행 한다.
- 진행전에 speckit.clarify 와 speckit.analyze 가 진행되어 있는지 확인후, 미 진행인 경우 사용자에게 안내 한다.
- tasks.md에서 병렬로 진행이 가능한 항목은 병렬로 진행하고 사용자에게 알린다.
- tasks.md의 주요 단위에서 (N.N, T\_<XXX>) 의 완료시, 해당 작업이 완료되면 완료문서를 생성하고, 해당 타스트를 완료체크 하고, 깃 커밋 반드시 하도록 지시한다.
- 문제점이 발생하는 경우에는 사용자에게 확인 받고 진행한다.
