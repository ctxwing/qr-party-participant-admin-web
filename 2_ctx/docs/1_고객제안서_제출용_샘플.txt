□ 프로젝트 제안 개요
- 귀사의 요구사항(Kmong 공고 및 첨부링크 등)을 면밀히 분석하였습니다.
- 태블릿 환경 최적화, 강력한 저작권 보호, 커스텀 필기가 가능한 악보 플랫폼 프로토타입을 준비하여 제안 드립니다.

□ 제안드리는 핵심 3가지 차별성
① 태블릿 펜슬 최적화 벡터 기반 필기 엔진 적용
- Fabric.js 기반의 고성능 엔진으로 태블릿 입력 지연을 최소화하고 매끄러운 필기감을 구현.
- 필기 데이터는 악보와는 별도로 JSON 벡터 형식으로 저장되어 기기 간 실시간 동기화가 가능하며, 원본 악보의 품질 손상 없이 자유로운 수정 및 삭제가 가능.

② 3중 저작권 보호 구조 및 동적 워터마크 시스템 구축
- Pre-signed URL을 통한 악보 원본 경로 은닉(제한시간 동안만 허용), 사용자 정보가 포함된 실시간 동적 워터마크, 등급별 인쇄 횟수 제한 기능을 통합 적용함.
- 무단 다운로드를 방지하고 출력물에도 추적 가능한 정보를 강제 삽입하여 저작권을 철저히 보호함.

③ PDF와 메모 캔버스간 분리형 렌더링 및 스마트 연습 도구 탑재
- 악보 PDF와 필기 레이어를 물리적으로 분리하여 로딩 성능을 확보하고, 악보 업데이트 시에도 기존 필기 데이터가 유지되는 유연한 구조를 설계함.
- 플랫폼 내 내장된 스마트 메트로놈과 연습 타이머를 통해 별도의 앱 없이도 원스톱 연습 환경을 제공하며 학습 데이터 축적 기반을 마련함.

□ 현재 프로토타입 기능 요약
- 핵심 뷰어 엔진(Zero-Scroll Layout): 디바이스 화면(모니터, 태블릿 등) 크기에 맞춰 스크롤 없이 악보가 한눈에 완벽히 들어오도록 자동 핏(Fit)을 맞춰주는 'Zero-Scroll' 레이아웃 기술을 적용
- 공통 및 뷰어모드: 고해상도 초고속 PDF 렌더링, 우측 사이드바 썸네일 기반의 빠른 페이지 탐색, 몰입감을 극대화하는 전체화면 모드를 딜레이 없이 제공.
- 스마트 메모 도구(Annotation Toolbar): 연주와 연습에 방해되지 않는 직관적인 플로팅 툴바 형태로 설계.
  * 다양한 드로잉: 색상 및 두께 제어가 가능한 자유형 펜, 중요 구간 표시를 위한 반투명 형광펜
  * 텍스트 및 편집: 식별이 쉬운 텍스트 메모 입력과 정밀한 지우개, 실행 취소(Undo) 및 화면 전체 지우기(Clear) 지원
  * 데이터 영속성: 작성 즉시 원본을 훼손하지 않고 개별 레이어로 분리되어 클라우드에 자동 저장(Auto-Save)되어 언제든 기기를 바꿔도 동일한 필기 결과를 유지.
- 보안 및 권한: 구독 등급별 접근 제어, 무단 캡처 방지용 동적 워터마크, 미구독자 API 원천 차단.
- 기술 스택: Next.js 16+(React 19), Strapi v5(Headless CMS), PostgreSQL, MinIO(S3), Docker 환경으로 모든 클라우드에 배포 가능.
