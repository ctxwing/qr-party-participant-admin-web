### 🔍 서비스별 상세 분석

#### 1\. Cloudflare R2 (추천 ⭐)

가장 추천하는 서비스입니다. 특히 데이터 전송료(Egress)가 없다는 점이 개발자들에게 가장 큰 메리트입니다. VSC에서 S3 확장 프로그램을 통해 연결하기 매우 쉽습니다.

  * **무료 한도:** 저장 용량 10GB, Class A 작업 100만 회, Class B 작업 1,000만 회/월.

#### 2\. Backblaze B2

오랫동안 신뢰받아온 서비스입니다. S3 호환 API를 공식 지원하며, 10GB까지는 신용카드 등록 없이도 시작할 수 있는 경우가 많아 테스트용으로 적합합니다.

  * **무료 한도:** 저장 용량 10GB 영구 무료.

#### 3\. Oracle Cloud (Always Free)

무료로 제공되는 용량이 20GB로 가장 넉넉합니다. 하지만 오라클 클라우드 특유의 복잡한 IAM 설정 때문에 초기 설정 시 러닝커브가 존재합니다.

  * **무료 한도:** 20GB 스토리지 및 월 10GB 데이터 전송.

#### 4\. MinIO (자체 구축형)

Michael 님의 **ctx-tower (Ubuntu 24.04)** 서버가 있으므로, Docker를 통해 직접 MinIO를 띄우는 것도 좋은 방법입니다. 이 경우 하드웨어 용량이 허락하는 한 무제한으로 S3 환경을 구축할 수 있습니다.

  * **특징:** S3와 API가 거의 100% 일치하여 학습 및 로컬 테스트용으로 최상입니다.
