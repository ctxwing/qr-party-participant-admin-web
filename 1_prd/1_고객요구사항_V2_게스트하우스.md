업무 내용
[프로젝트 개요]
- 게스트하우스 파티용 실시간 소통 반응형 웹 개발

[프로젝트 배경 및 목표]
- 본 프로젝트는 게스트하우스에서 주최하는 파티의 참석자 간 원활한 소통과 이벤트 진행의 편의성 증대를 목표로 합니다. 참석자들이 QR코드를 통해 파티 전용 웹페이지에 접속하여 실시간으로 메시지를 주고받고, 호감도를 표현하며, 노래 신청 등 다양한 상호작용을 할 수 있는 반응형 웹 서비스를 개발하고자 합니다. 이를 통해 파티의 재미를 극대화하고, 관리자는 파티 현황을 효율적으로 파악하고 관리할 수 있는 시스템을 구축하는 것이 핵심입니다.

[과업 범위]
1. 수행 범위
- 상세 기획: 요구 사항 정의, 기능 정의, 화면 설계, 정책 설계
- UI/UX 디자인
- 프런트엔드/Client 개발 (PC/Mobile Web 반응형)
- 백엔드 개발
- 서버/DB/인프라 구성

2. 상세 기능 요구 사항
2-1. 사용자 페이지 기능
- QR코드를 통한 파티 입장 및 인증
- 실시간 파티 진행 시간 및 남은 시간 표시
- 현재 파티 참여 인원 수 실시간 표시
- 본인의 파티 참여 정보 확인 (예: 1차/2차 파티 신청 현황)
- 닉네임 설정 및 변경 (최대 3회 제한)
- 개인 현황판: 수신한 큐피트, 쪽지, 호감도 개수 확인 및 알림 기능
- 다른 참석자에게 쪽지 발송 기능
- 큐피트 발사 기능 (참여자당 2회 제한)
- 호감도 표현 기능 (참여자당 3회 제한)
- 실시간 호감도 순위 확인
- 관리자에게 노래 또는 SOS 요청 기능
- 파티에서 나가기 기능

2-2. 관리자 페이지 기능
- 파티 참석자(유저) 목록 조회 및 관리
- 기존 홈페이지 예약 내역 연동 (※ 계약 시 개발 가능 여부 및 범위 협의)
- 전체 참석자 대상 실시간 공지 발송 기능 (채팅방 형태)
- 사용자 활동 실시간 모니터링 (쪽지, 호감도, 닉네임 변경 이력 등)

[지원 디바이스 ]
