# [참여자 모바일 UX 설계] 20년 경력 모바일 UX 전문가의 설계안

**설계 철학**: "현장의 흥분을 방해하지 않는 매끄러운 연결"
파티 현장은 어둡고 소란스러우며, 사용자는 한 손으로 기기를 조작할 확률이 높습니다. 따라서 **Glanceability(가시성)**, **One-thumb Operation(한 손 조작)**, **Emotional Feedback(감성적 피드백)**을 핵심 가치로 설계합니다.

---

## 1. 주요 화면 설계

### [M-01] 퀵 진입 및 닉네임 설정 (Entry)
- **UI 구조**:
    - 중앙: "파티에 오신 것을 환영합니다!" (대형 타이포그래피)
    - 입력창: 닉네임 입력 (플레이스홀더: "현장에서 불릴 닉네임을 입력하세요")
    - 하단: [파티 시작하기] (전폭 버튼, 엄지 손가락 위치 고려)
- **UX 포인트**:
    - **초스피드 진입**: 별도 로그인 없이 즉시 입력창에 포커스.
    - **변경 횟수 노출**: "남은 변경 횟수: 3회"를 입력창 하단에 작게 표시하여 신중한 선택 유도.

### [M-02] 메인 대시보드 (Main Hub)
- **UI 구조**:
    - 상단: 파티 남은 시간 카운트다운 (글래스모피즘 효과, 시인성 확보)
    - 중앙 카드: '나의 현황' (쪽지 📩, 큐피트 💘, 호감도 ⭐ 개수 노출)
    - 하단 리스트: '지금 핫한 파티원' (실시간 상호작용이 많은 유저 3인 프리뷰)
- **UX 포인트**:
    - **실시간 피드백**: 쪽지나 호감도를 받으면 카드 숫자가 부드러운 애니메이션과 함께 카운팅 업.

### [M-03] 참여자 탐색 및 상호작용 (Discovery)
- **UI 구조**:
    - 상단: 검색 바 (닉네임 검색)
    - 리스트: 참여자 카드 (닉네임, 현재 기분/상태 한 줄 문구)
    - 카드 클릭 시: 상호작용 팝업 (쪽지 보내기, 큐피트 발사, 호감도 주기)
- **UX 포인트**:
    - **직관적 행동**: 각 상호작용 버튼에 고유한 아이콘과 테마 컬러 부여 (큐피트: 핑크, 호감도: 옐로우).

### [M-04] 실시간 랭킹 보드 (Leaderboard)
- **UI 구조**:
    - 1~3위: 대형 프로필 아이콘과 왕관 이미지 (React Bits 애니메이션 적용)
    - 4위 이하는 스크롤 가능한 리스트 형태.
- **UX 포인트**:
    - **역동성**: 순위 변동 시 실시간으로 위치가 바뀌는 부드러운 트랜지션 적용.

### [M-05] 전역 플로팅 버튼 (Global FAB)
- **UI 구조**:
    - 우측 하단: [+] 아이콘의 원형 버튼.
    - 클릭 시: [SOS 요청], [노래 신청] 메뉴가 부채꼴 모양으로 확장.
- **UX 포인트**:
    - **긴급성**: SOS 요청은 붉은색 테두리로 강조하여 실수 방지 및 시인성 확보.
