# QR Party Participant Admin - 기동 가이드

본 문서는 프로젝트의 로컬 개발 서버 기동 방법 및 관리자 접속 정보를 안내합니다.

## 1. 환경 설정 및 준비

### 1.1 필수 도구
- **Runtime**: Bun (v1.0 이상 권장)
- **Framework**: Next.js 16

### 1.2 의존성 설치
서버를 처음 기동하거나 패키지 변경 시 실행하십시오.
```bash
cd prj_source/frontend
bun install
```

## 2. 서버 기동 방법

### 2.1 통합 스크립트 사용 (권장)
루트 디렉토리에 있는 `start_frontend.sh`를 실행하면 포트 충돌 확인 및 서버 기동이 자동으로 진행됩니다.
- **포트**: `59500`
- **실행**:
```bash
bash start_frontend.sh
```

### 2.2 수동 기동
```bash
cd prj_source/frontend
bun run dev --port 59500
```
