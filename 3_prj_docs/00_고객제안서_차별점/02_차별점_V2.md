# 제안 차별점 (Key Differentiators)

## 1. 현장 맞춤형 파티 UX 최적화 설계
*   **차별성**: 파티 현장의 어둡고 소란스러운 환경을 고민하여 참여자의 즉각적 인지를 할 목적으로 Glassmorphism 다크 테마와 한 손 조작 기반 UI, Vibration(진동) 피드백을 반영하여 별도 안내 없이도 직관적 조작이 가능한 효과를 도출함
*   **효용**: 익숙하지 않은 참여자도 QR 스캔 직후 즉시 서비스 이용이 가능하며, 관리자는 교육 없이도 파티 현장 제어가 가능함

## 2. 브랜디드 QR 및 일괄 횟수 지급 관리
*   **차별성**: 현장 인쇄물 제작의 번거로움과 이벤트 진행의 유연성을 고민하여 QR 코드 커스터마이징(색상/해상도 PNG·SVG)과 전체 참여자 대상 일괄 횟수 지급(Bulk Grant) 기능을 반영하여 외부 툴 없이 현장에서 즉시 운영 준비가 완료되는 효과를 도출함
*   **효용**: 파티 컨셉에 맞춘 QR 인쇄물을 즉시 제작할 수 있고, 이벤트 타임에 모든 참여자에게 동시 횟수 부여로 파티 흐름이 끊기지 않음

## 3. 초저지연 실시간 동기화 엔진
*   **차별성**: 대규모 파티에서의 데이터 지연을 고민하여 참여자 상호작용(쪽지, 큐피트, 호감도)과 관리자 공지를 1초 내 동기화할 목적으로 Supabase Realtime Broadcast 채널을 반영하여 폴링(Polling) 방식 대비 현저히 낮은 지연 체감 효과를 도출함
*   **효용**: 쪽지와 호감도가 즉시 반영되어 파티 몰입감이 유지되며, 관리자 공지가 전 참여자에게 동시 팝업으로 전달됨

## 4. 대용량 데이터 처리 및 서버 사이드 보안 아키텍처
*   **차별성**: 장기 운영 시 데이터 누적에 따른 성능 저하와 관리자 정보 유출을 고민하여 Ag-Grid v35 최적화 인덱싱과 Next.js Middleware 기반 서버 사이드 인증(Better Auth)을 반영하여 1만 건 이상에서도 0.5초 내 조회 및 비인가 접근 원천 차단 효과를 도출함
*   **효용**: 파티 누적에도 관리자 페이지 응답 속도가 일정하게 유지되며, 관리자 전용 데이터(전체 쪽지, 참여자 정보 등)의 외부 노출을 방지함

## 5. 파티 이벤트 도메인 경험 기반 선제적 엣지케이스 대응
*   **차별성**: 닉네임 도용, 횟수 소진, 세션 끊김 등 실제 파티 운영에서 빈번히 발생하는 이슈를 고민하여 닉네임 변경 3회 제한, 상호작용 쿨타임(Rate Limit), 세션 복구 로직을 선제적으로 반영하여 운영 중 돌발 상황을 최소화하는 효과를 도출함
*   **효용**: 고객이 예상하지 못한 운영 리스크를 시스템 차원에서 사전 차단하여 안정적인 파티 진행이 가능함
