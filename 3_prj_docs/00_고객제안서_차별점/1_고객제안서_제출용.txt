- 특히 대규모 인원이 동시에 접속하는 환경에서의 안정적인 서버 운영 노하우를 갖추고 있습니다.

□ 구현 검증 및 산출물
- 저희 팀은 개발 구현 가능성과 성능 검증을 위해 핵심 기능을 프로토타입으로 개발하여 함께 제안 드립니다.
- 실제 작동하는 1만 건 이상의 파티 이력 대응 목록, 실시간 공지 발송, 브랜디드 QR 생성 기능을 직접 확인하실 수 있습니다.

□ 프로젝트 제안 개요
- 귀사의 요구사항(Wishket 공고 및 상세 내용)을 면밀히 분석하였습니다.
- 10일 이내의 짧은 납기 내에 안정적인 고성능 관리 시스템과 참여자용 모바일웹을 통합 제공하는 프로토타입을 준비하여 제안 드립니다.

□ 제안드리는 3가지 핵심 차별성
① 운영 현장 최적화 브랜디드 QR 관리 및 다운로드 시스템
- 단순 접속용 QR을 넘어 관리자가 현장 분위기에 맞춰 QR 색상을 실시간 변경하고 고해상도(PNG/SVG)로 즉시 다운로드할 수 있는 전용 도구를 반영함.
- 외부 디자인 툴 없이도 즉각적인 브랜딩 및 인쇄물 제작이 가능하여 현장 운영 준비 시간을 혁신적으로 단축함.

② Supabase Realtime 기반 초저지연 실시간 공지 엔진
- 참여자의 주의를 즉각 환기할 목적으로 방송(Broadcast) 기능을 반영하여 1초 이내의 초저지연 실시간 팝업 공지 효과를 도출함.
- 파티의 흐름을 끊지 않고도 관리자가 긴급 안내나 이벤트를 참여자 전체에게 즉각 전달하여 현장 장악력을 극대화함.

③ 엔터프라이즈급 고성능 데이터 처리 및 가이드 UI 적용
- 1만 건 이상의 데이터 환경에서도 0.5초 이내의 조회 속도를 보장하는 Ag-Grid v35 최적화 설계와 파티 생명주기를 시각화한 가이드 UI를 반영함.
- 장기 운영 시에도 시스템 느려짐이 없으며, 신규 운영자도 별도 교육 없이 즉시 파티 운영이 가능한 편의성을 제공함.

□ 현재 프로토타입 기능 요약
- 관리자 시스템: 파티 생명주기 관리(생성/시작/종료), 전체 쪽지 조회, 닉네임 변경 로그, 실시간 공지 발송 시스템 완비.
- 데이터 및 보안: 1만 건 이상의 대용량 데이터 렌더링 최적화, Next.js Middleware 기반 서버 사이드 경로 보호 적용.
- 기술 스택: Bun 1.1+, Next.js 16.2.1+, Supabase(PostgreSQL), Drizzle ORM, Ag-Grid v35, Better Auth.

□ 보유 경험
- 반응형 PC/모바일웹 통합 구축 역량 (Next.js 전문가 그룹)
- 실시간 데이터 스트리밍 및 동기화 시스템 개발 경험 보유
- 고성능 데이터 그리드(Ag-Grid) 및 대시보드 커스터마이징 역량
- Docker 기반 컨테이너화 배포 및 클라우드(AWS/Supabase) 운영 노하우

□ 총 개발 기간은 10일 이내 완료 가능합니다. (상용 배포 및 마이그레이션 포함)

1단계: 요구사항 상세 분석 및 UI/UX 테마 설정 (2일)
2단계: 파티 관리 로직 및 실시간 상호작용 엔진 구축 (3일)
3단계: QR 브랜딩 및 관리자 보안 미들웨어 강화 (2일)
4단계: 통합 테스트 및 성능 최적화 (2일)

□ 추가 제공 가치
- 최신 Next.js 16 기반의 고성능 아키텍처로 최상의 사용자 경험 보장.
- 서버 사이드 보안 설계를 통해 관리자 권한 및 민감 데이터 유출 원천 차단.
- 파티 이력 10,000건 이상에도 성능 저하 없는 안정적인 확장성 제공.
