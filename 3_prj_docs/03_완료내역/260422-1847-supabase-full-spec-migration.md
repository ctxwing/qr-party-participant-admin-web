- **수동 설치**: 시스템 정책으로 인한 NPM 글로벌 설치 제한을 우회하여 GitHub 바이너리 직접 설치 수행.
- **PAT 연동**: Personal Access Token을 활용하여 개발 서버의 포트 제한(5432)을 우회하고 Management API를 통해 DB 관리 가능하도록 설정.
- **가이드 작성**: `3_prj_docs/Supabase적용/SUPABASE_CLI_GUIDE.md` 및 `SUPABASE_SETUP_GUIDE.md` 업데이트.

### 2.2 DB 스키마 마이그레이션 (Full Spec 반영)
- **참여자 테이블**: `is_first_applied`, `is_second_applied` (신청 상태), `cupid_count`, `like_count` (잔여 횟수) 필드 추가.
- **상호작용 테이블**: `interactions` 테이블 신규 생성 및 참여자/세션 외래키 연결.
- **Realtime 설정**: `interactions` 테이블에 대한 실시간 데이터 변경 구독 활성화.

### 2.3 대시보드 UI 및 기능 고도화
- **신청 상태 표시**: 참여자의 1차/2차 신청 완료 여부를 상단에 시각적으로 표시.
- **상호작용 로직**: 큐피트 2회, 호감도 3회 제한 로직 및 DB 차감 연동.
- **실시간 통계**: 내가 받은 쪽지, 큐피트, 호감도 수를 실시간으로 집계하여 표시.
- **가상 데이터**: 명세서 규격에 맞춘 가상 유저 20명 재생성.
